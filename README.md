# Velox Tech

Missão
Desenvolver soluções digitais essenciais, fornecendo a base técnica especializada que os empreendedores precisam para validar suas ideias de negócio de forma rápida, eficiente e acessível, transformando conceitos em realidade funcional.

PropósitoSer o catalisador tecnológico para a próxima geração de empreendedores, conectando a visão de negócio à execução técnica de excelência. Existimos para acelerar sonhos e garantir que uma grande ideia encontre seu primeiro passo digital com segurança e solidez.

Manifesto "O Ponto de Partida
Nós Acreditamos no Potencial: Acreditamos que toda grande ideia merece ter uma chance de sair do papel. Somos a ponte que transforma a estratégia de negócios em uma solução digital robusta.
Somos Parceiros, Não Apenas Fornecedores: Nossa experiência de mais de 10 anos não serve apenas para escrever código, mas para traduzir a sua visão. Sentamos ao seu lado, entendemos a dor do seu cliente e criamos juntos a solução mais enxuta e inteligente.
Seu primeiro passo precisa ser leve. Não vamos construir um castelo quando você só precisa de uma fundação sólida. Nosso foco é validar, aprender e crescer a partir de um MVP que realmente funcione.
Falamos a sua língua, explicamos o processo e garantimos que você se sinta no controle de cada etapa da evolução digital do seu negócio.
Se o primeiro MVP não for perfeito, ele nos dará dados valiosos. Nosso compromisso é com o aprendizado contínuo e a evolução do produto, até que ele encontre o ajuste perfeito no mercado.
Sua Experiência, Nossa Tecnologia. Juntos, o Negócio.

##  How to run

All commands are run from the root of the project, from a terminal:

| Command                | Action                                             |
| :--------------------- | :------------------------------------------------- |
| `npm install`          | Installs dependencies                              |
| `npm run dev`          | Starts local dev server at `localhost:3000`        |
| `npm run build`        | Build your production site to `./dist/`            |
| `npm run preview`      | Preview your build locally, before deploying       |
